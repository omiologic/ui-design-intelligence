# Changelog

All notable changes to `ui-react-component-skills` will be documented in this
file.

## 0.9.0 - Sprint 011 Initial Bundle

- Added the `ui-react-component-skills` bundle manifest and README.
- Included six React component skills:
  - `mockup-to-component-analysis`
  - `component-spec-to-react-plan`
  - `storybook-from-component-spec`
  - `shared-ui-component-planner`
  - `app-specific-component-planner`
  - `react-component-review`
- Registered `ComponentSpec` contracts, schema, vocabulary, templates, pattern
  guidance, config docs, and example assets.
- Deferred `ui-shared-component-skills` and `ui-app-component-skills` as future
  bundle splits until the standalone React component workflow stabilizes.
