# Header Foundation Method

## Decision Heuristics

Choose header behavior from navigation complexity and conversion priority.

## Anti-Pattern

Do not assume sticky headers or mega menus without evidence or product need.

## Worked Example

For six primary links and an appointment CTA, use desktop
`logo-left nav-center actions-right` and mobile `logo-left menu-button-right`
with drawer navigation.

Return JSON compatible with `header-foundation.schema.json`.
