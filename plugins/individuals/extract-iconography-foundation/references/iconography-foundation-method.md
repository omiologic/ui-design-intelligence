# Iconography Foundation Method

## Decision Heuristics

Prefer one consistent outline style for early seeds unless source evidence
requires another style.

## Anti-Pattern

Do not specify custom icon artwork or production SVG paths in the seed.

## Worked Example

Recommend `lucide-react`, outline style, `1.75px` stroke, rounded corners, and
rules that icons support labels rather than replace them.

## Hand-Off

Return JSON compatible with `iconography-foundation.schema.json`.
