# React Component Skills

This directory documents the React component skill layer inside
`ui-design-intelligence`.

The layer is skill-oriented. It owns component analysis, `ComponentSpec`,
implementation planning, Storybook planning, shared/app boundary decisions, and
React component review. It does not own a React runtime component library,
Storybook application, visual regression infrastructure, npm package, or demo
app.

Core docs:

- `ecosystem-architecture.md`
- `recommended-file-organization.md`
- `repository-placement-recommendation.md`
- `component-build-config.md`

The installable bundle is `ui-react-component-skills`.
